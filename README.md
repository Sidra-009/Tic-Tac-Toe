# 🎮 Tic Tac Tao 
### **“An unbeatable Tic-Tac-Toe AI powered by Minimax and Alpha-Beta Pruning.”**

TicTac AI is an intelligent Tic-Tac-Toe game built using **Python** and **Pygame**, designed for demonstrating AI concepts such as **Minimax** and **Alpha-Beta Pruning**.  
The AI evaluates game states, predicts outcomes, blocks human wins, and always chooses the optimal move — making it impossible to defeat.

---

## 🚀 Overview  
The power of **game-tree search**, **decision making**, and **optimization techniques** used in AI.  
TicTac AI transforms the classic X-O game into a strategic and fully deterministic experience.

---

## 💡 Key Features  
- 🎮 Human vs AI gameplay  
- 🤖 Minimax algorithm (complete implementation)  
- ✂️ Alpha-Beta pruning for efficiency  
- 🎨 Clean & centered UI using Pygame  
- 🔊 Optional sound effects  
- ⚡ Fast, optimized move calculation  
- 🧠 AI that never loses  

---

## 🧠 How the AI Works  

### **Minimax Algorithm**
- Evaluates all future game states  
- Maximizes AI’s winning chances  
- Minimizes human player’s advantage  

### **Alpha-Beta Pruning**
- Removes unnecessary branches  
- Speeds up decision-making  
- Reduces computation time  

**Result:** A perfectly optimal and unbeatable AI.

- Pygame library

```bash

cd ttt-ai

# 2️⃣ Install pygame

# 3️⃣ Run the game
python main.py
```
